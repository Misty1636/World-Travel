## 簡介
此作品的功能有:
+ 前台: 商品導覽、商品一覽、購物車、商品購買、優惠券套用、訂單結帳
+ 後台: 商品管理、簡易訂單統計、優惠券管理
## 主要技術
+ Vue Cli 3
+ VueRouter
+ Vuex管理
+ Webpack
+ RWD(支援到iphone5 320px)
+ Bootstrap 4 + 大量手刻SCSS
+ Ajax串接API
+ ESlint(Airbnb)提升程式碼品質
## 使用的Plugin
